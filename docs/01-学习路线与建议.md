## 一、这本书教什么

一句话概括：**用 PyTorch 从零手写一个类 ChatGPT 的大语言模型**。

全书按"俄罗斯套娃"结构展开——每一章都在前一章的基础上叠加新代码，最终得到一个完整的、可训练、可微调的小型 GPT 模型（124M 参数，GPT-2 规模）。

```
第1章(理论) → 第2章(分词+嵌入) → 第3章(注意力) → 第4章(GPT模型)
    → 第5章(预训练) → 第6章(分类微调) → 第7章(指令微调)
```

### 与其他教程的区别

| 特性 | 本书 | 常见教程 |
|------|------|---------|
| 依赖 | 仅 PyTorch + tiktoken | HuggingFace Transformers 等高层库 |
| 实现方式 | 从零手写每一行 | 调包 `model = AutoModel.from_pretrained()` |
| 学习深度 | 理解每个张量形状变化 | 停留在 API 调用层面 |
| 模型规模 | 124M（可在笔记本上跑） | 通常直接加载 7B+ 预训练模型 |

## 二、前置知识自查

在开始之前，确认你具备以下基础：

| 要求 | 程度 | 怎么办 |
|------|------|--------|
| Python 编程 | ⭐⭐⭐⭐⭐ 必须扎实 | 能写类、函数、列表推导、文件操作 |
| PyTorch 基础 | ⭐⭐⭐ 建议了解 | 先看附录 A（2天） |
| 深度学习概念 | ⭐⭐ 有帮助 | 知道前向传播、反向传播、损失函数即可 |
| 数学 | ⭐⭐ 够用就行 | 矩阵乘法、Softmax、交叉熵——书中会讲 |

**最低门槛**：Python 熟练 + 愿意学。

## 三、四阶段学习路线

### 🟢 阶段一：基础入门（第 1-2 周）

```
目标：跑通文本处理全流程，理解注意力机制
```

| 步骤 | 内容 | 关键输出 | 建议时长 |
|------|------|---------|---------|
| 0 | 环境配置 | `pip install - requirements.txt` | 30 分钟 |
| 1 | 附录 A：PyTorch 速成 | 理解 Tensor、Autograd、nn.Module | 1-2 天 |
| 2 | 第 1 章：理解 LLM | 了解 GPT 架构全貌（纯理论） | 半天 |
| 3 | 第 2 章：文本处理 | 实现 BPE 分词器 + DataLoader | 2-3 天 |
| 4 | 第 3 章：注意力机制 | 理解 QKV + 因果掩码的每步张量变化 | 3-4 天 |

> ⚠️ **第 3 章是全书最抽象的部分**。务必用手画出每一步 `(batch, seq_len, dim)` 的形状变化。如果你能闭眼说出 attention 的张量变换流程，就过关了。

### 🟡 阶段二：核心模型（第 3-5 周）

```
目标：独立实现完整的 GPT 模型并进行预训练
```

| 步骤 | 内容 | 关键输出 | 建议时长 |
|------|------|---------|---------|
| 5 | 第 4 章：GPT 模型 | `gpt.py`（277 行完整实现） | 4-5 天 |
| 6 | 第 5 章：预训练 | 完整训练循环 + 文本生成 | 5-7 天 |

> 🔥 **关键挑战**：第 4 章的 `gpt.py` 不要只看——关掉屏幕，自己从空白文件写出来。遇到张量形状错误是学习过程的一部分。

> 💻 **硬件**：第 5 章训练 124M 模型可能需要数小时。Google Colab 免费 T4 GPU 即可胜任。

### 🔵 阶段三：微调实践（第 6-8 周）

```
目标：掌握两种核心微调范式
```

| 步骤 | 内容 | 关键输出 | 建议时长 |
|------|------|---------|---------|
| 7 | 第 6 章：分类微调 | 垃圾邮件检测器 | 2-3 天 |
| 8 | 第 7 章：指令微调 | 对话模型 | 4-5 天 |
| 9 | 附录 E：LoRA | 参数高效微调 | 1-2 天 |

### 🟣 阶段四：深入探索（长期）

学完主线后，按兴趣方向选择：

```
🔧 推理优化        🏗️ 架构变体          🎯 注意力进阶
├─ KV Cache        ├─ GPT→Llama 转换     ├─ GQA（分组查询注意力）
├─ 训练加速        ├─ Qwen3 从零实现     ├─ MLA（多头潜在注意力）
└─ 混合专家(MoE)   ├─ Gemma 3 从零实现   └─ SWA（滑动窗口注意力）

🤖 模型对齐        🧠 续作
├─ DPO 对齐        └─ GRPO 强化学习
└─ 数据集生成         └─ 推理时扩展
```

## 六、常见问题

**Q: 没有 GPU 能学吗？**
A: 能。第 2-4 章完全不需要 GPU。第 5 章训练建议用 Colab 免费 T4。

**Q: 需要买书吗？**
A: 仓库代码免费，但书中的图示和讲解是代码无法替代的，强烈推荐。

**Q: 多久能学完？**
A: 全职学习约 6-8 周完成主线 7 章。业余约 3-4 个月。

**Q: 学完能达到什么水平？**
A: 能独立实现 Transformer/GPT 架构，理解预训练+微调完整流程，看懂主流 LLM 论文的架构部分。
