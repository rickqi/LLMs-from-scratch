## 关键发现

1. **30 epochs 有效但不够**: 波动率 RankIC 从 -0.0248 → +0.0460（转正），但仍远低 v1 (+0.569)
2. **股票池扩大稀释信号**: 183 只中大量新 IPO 股票（2022+上市）与训练集分布不同
3. **最优参数漂移**: v1 偏好 top_k=5（集中），v2 偏好 top_k=20（分散）
4. **信号 Sharpe 稳定**: 尽管 RankIC 恶化，信号回测 Sharpe 仍维持在 1.1-1.4
