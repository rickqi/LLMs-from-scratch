## 可执行方案

| # | 方向 | 思路 | 预期收益 | 工作量 |
|---|------|------|---------|--------|
| 1 🔥 | **波动率回测验证** | 滚动窗口验证预测波动率 vs 实际波动率 | 确认模型可靠性 | 1h |
| 2 🔥 | **双信号集成** | 波动率(低)+方向(涨)=买入; 波动率(高)+方向(跌)=卖出 | 信号质量↑ | 1h |
| 3 | **每日自动化流水线** | Tushare拉最新数据→预测→保存→推送 | 生产就绪 | 2h |
| 4 | **长历史全量重训** | 单独为54只长历史半导体训波动率模型 | RankIC +0.02~0.05 | 1h |
| 5 | **特征重要性分析** | 逐个移除OHLCV特征，找最小有效集 | 模型简化 | 1h |

## 不建议

| 行动 | 理由 |
|------|------|
| 扩大品种到CSI300 | 15次实验均负RankIC |
| BSQ/复杂架构 | LSTM已统治性领先 |
| 日内高频 | Tushare免费限2次/天 |
| 更多特征(PE/PB等) | 实验证明不如纯OHLCV |

## 推荐执行顺序

```
1 → 波动率回测 (确认RankIC=+0.569在实际时间序列上有效)
2 → 双信号集成 (波动率×方向 = 更强信号)
3 → 自动化流水线 (每日拉数据→预测→保存)
```
