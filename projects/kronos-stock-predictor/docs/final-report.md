# Kronos Stock Predictor — 最终评估报告

> 2026-06-18 | 全部实验完成

## 实验总结

| 阶段 | 方法 | 结果 | 可靠性 |
|------|------|------|--------|
| 初始 | 静态 split (68只) | Sharpe 2.28 | ⚠️ 过拟合 |
| 扩展 | 静态 split (183只) | Sharpe 2.35 | ⚠️ 过拟合 |
| 验证 | 季度 Walk-forward | RankIC -0.021 | ✅ 真实 |
| 细化 | 周度 Walk-forward | RankIC -0.218 | ✅ 真实 |
| 翻转 | 反向信号 | Sharpe -0.61 | ✅ 确认无效 |

## 核心结论

```
LSTM 波动率预测在 A 股半导体板块不具有稳定的样本外预测能力。

静态 split 的高 Sharpe 是参数过拟合 + 数据窥探偏差的产物。
Walk-forward 诚实评估暴露了真实上限：模型 ≈ 随机。
```

## 根因

1. **市场结构性变化**: 2025-2026 年半导体板块走势与 2018-2024 年模式显著不同
2. **低信噪比**: 金融时间序列天然信噪比极低
3. **模型容量不足**: LSTM 128h 可能不足以捕捉复杂市场模式
4. **特征维度有限**: 仅 6 维 OHLCV，缺少更多市场微观结构信息
