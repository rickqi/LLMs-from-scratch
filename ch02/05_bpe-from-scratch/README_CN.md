# 从零开始的字节对编码（Byte Pair Encoding, BPE）分词器

- [bpe-from-scratch-simple.ipynb](bpe-from-scratch-simple.ipynb) 包含可选（奖励）代码，用于解释和展示 BPE 分词器（BPE tokenizer）在底层如何工作；这个版本注重简单性和可读性。

- [bpe-from-scratch.ipynb](bpe-from-scratch.ipynb) 实现了一个更复杂（也更复杂）的 BPE 分词器，它在所有边缘情况下与 tiktoken 表现相似；它还具有加载官方 GPT-2 词汇表（vocabulary）的附加功能。
